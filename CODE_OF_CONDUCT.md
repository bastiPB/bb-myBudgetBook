# Code of Conduct

## Ziel
Wir wollen eine freundliche, respektvolle und produktive Umgebung schaffen – für Nutzer:innen und Beitragende gleichermaßen.

## Unsere Erwartungen
Bitte:
- sei respektvoll und konstruktiv
- formuliere Kritik sachlich (Problem/Beobachtung/Vorschlag)
- respektiere unterschiedliche Erfahrungslevel
- halte Diskussionen themenbezogen
- gehe sorgfältig mit privaten Informationen um (keine doxxing/Leaks)

## Unzulässiges Verhalten
Nicht ok sind u. a.:
- Beleidigungen, Herabwürdigungen, gezielte Provokationen
- Belästigung oder Einschüchterung
- Veröffentlichung privater Informationen anderer (ohne Erlaubnis)
- Spam, Trolling, oder absichtliches Stören von Diskussionen

## Geltungsbereich
Gilt für alle Projektbereiche, z. B.:
- GitHub Issues, Pull Requests, Discussions
- Commit-Kommentare, Review-Kommentare
- Projektkommunikation im Zusammenhang mit diesem Repo

## Durchsetzung
Maintainer können Beiträge moderieren (z. B. Kommentare entfernen, Issues schließen, Accounts blockieren), wenn Verhalten diesem Code widerspricht.
