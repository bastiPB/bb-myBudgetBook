# Roadmap

Siehe docs/01-mvp.md

## v0.2 (kandidaten)
- Budgets (Urlaub)
- Sparpläne
- Einnahmen/Ausgaben Tracking
- Kategorien/Tags
- Export (CSV/JSON)
- Mehr Reports/Charts
- **Kontext-Modell (Spaces)**: persönliche + geteilte Bereiche für Familien (Details: docs/13-rbac-vision.md)
- **Admin-Einstellung: Selbst-Registrierung ein/aus** — Toggle ob `/register` offen ist oder User nur vom Admin angelegt werden (Details: docs/02-ideas.md)
- **Passwort ändern** — User kann sein eigenes Passwort ändern (nützlich nach Admin-Erstellung mit vorgegebenem Passwort)
